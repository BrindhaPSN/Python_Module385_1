# Python Revision
- to run python script `python <filename.ext>` or `python3 <filename.ext>`

## Quick keys:
- Open and close terminal `ctrl+~`
- clearing/cleaning terminal: type `clear` or `ctrl + l`
- to view previous terminal commands cycle `up-arrow` or `downarrow`

## Built-in functions:
- print() - to show the data/result in the terminal/console
- input() - to get data from the user from console
